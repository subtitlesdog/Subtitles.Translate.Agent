# agent : Step1_DirectorAgent

💡 它是谁？：AI 翻译总导演
请稍候，我们的“总导演”正在看片... 🎬

在开始逐句翻译之前，我们需要先“读懂”这部片子。这个 Agent 就是我们请来的资深语言专家。

它正在为您做什么？

🧐 看类型：它在判断这是新闻、电影、教程还是搞笑视频，以免把严肃的新闻翻成搞笑段子。

🎭 定调子：它在决定说话的方式。是该文雅一点？还是口语化一点？是叫“亲爱的”还是叫“您”？

📖 理剧情：它在快速浏览全篇，搞清楚故事的来龙去脉，防止翻译到后面忘了前面。

它的绝招： 它就像给后续的翻译 AI 吃了一颗“聪明药”。有了它的分析，翻译出来的字幕将不再是冷冰冰的文字，而是有温度、有性格、有灵魂的对白。

# agent : Step2_GlossaryAgent
🛡️ Agent 名称：术语一致性守卫 (Terminology Consistency Guardian)
“翻译最大的灾难不是翻错，而是翻得不统一。本 Agent 将从源头扼杀这一隐患。”

功能简介： 在翻译真正开始之前，这位 Agent 会像一位严谨的“档案管理员”，对视频文本进行深度扫描。它不关心句子结构，它只关心实体（Entities）。它将构建一个专属的**“微型知识图谱”**，强制后续所有的翻译步骤遵守统一的标准。

它为何强大？ 普通翻译经常会犯“名字翻来覆去变”、“男人被翻成她”、“专业术语大白话”的低级错误。本 Agent 通过以下技术解决这些痛点：

🔍 智能实体消歧：它能识别 Bill 和 William 是同一个人，并将其映射为同一个中文译名，确保主角名字不会“变来变去”。

⚖️ 动态语域对齐：承接第一步的风格指南，它知道在“古风剧”里要把 Teacher 翻成“师尊”而不是“老师”，在“黑帮片”里要把 Boss 翻成“老大”而不是“老板”。

⚧️ 语境性别推断：告别 AI 常见的“性别混乱”。它根据上下文锁定角色性别，彻底解决 Him/Her 乱用的问题。
