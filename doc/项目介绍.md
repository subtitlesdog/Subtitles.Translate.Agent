# 亮点：
- 自动检测到文件编码
- 支持自定义ai大模型配置
- 支持为每个agent自定义不同端点
- 意群跨行（Enjambment）
- 多行滑动翻译

# 核心六个agent 
- Step1_DirectorAgent 全局理解
- Step2_GlossaryAgent 术语提取
- Step3_TranslatorAgent 翻译
- Step4_ReviewerAgent 质量审核 （暂未开源）
- Step5_PolisherAgent 润色优化 （暂未开源）
- Step6_TimingAdjusterAgent 时间轴优化 （暂未开源）
